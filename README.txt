MUSIC SEARCH BOT
================

Что делает:
- Принимает текстовый запрос (название песни или исполнителя).
- Ищет по сайту rus.hitmotop.com (через DuckDuckGo) и показывает топ-5 совпадений.
- По нажатию на найденный трек скачивает mp3 (если на странице есть прямая ссылка) и отправляет в личный чат.

ВАЖНО — юридическая оговорка:
- Этот бот предназначен только в том случае, если у тебя есть права на распространение треков.
- Не используй его для нарушения авторских прав.

Файлы в архиве:
- bot.py           — основной код бота
- requirements.txt — зависимости Python
- README.txt       — эта инструкция

Локальный запуск:
1) Установи Python 3.9+ и создай виртуальное окружение (рекомендуется).
2) Установи зависимости:
   pip install -r requirements.txt
3) Экспортируй переменную окружения с токеном:
   (Linux/macOS)
   export BOT_TOKEN="your_token_here"
   (Windows PowerShell)
   $env:BOT_TOKEN="your_token_here"
4) Запусти бота:
   python bot.py

Развёртывание на Render.com:
1) Создай репозиторий на GitHub и залей файлы из этого архива.
2) Создай Web Service на Render и подключи репозиторий.
   - Build Command: pip install -r requirements.txt
   - Start Command: python bot.py
3) В Render → Environment → Environment Variables добавь:
   BOT_TOKEN = твой_токен_из_BotFather
4) Деплой — Render установит зависимости и запустит контейнер.
   - Примечание: Render может "спать" на бесплатном тарифе. Для критичного сервиса рассмотрите платный тариф или пинги.

Отладка:
- Если бот не отвечает — смотри логи Render.
- Если не находились треки — возможно сайт изменил структуру и нужно адаптировать функции search_hitmotop / extract_song_info в bot.py.

Удачи!
